适用于新华三、华为交换机，多线程采集配置

本机需要能够SSH直连交换机

运行环境： Python 3.10 +

第一次运行请双击 “安装脚本执行环境”，安装相关组件


请在 “config”文件内修改需要采集的信息，如 dis version，默认即可使用


请在 “ip.xls” 里添加需要被执行的设备IP地址，以及第二列第三列的对应交换机账户密码

在第四列填入需要被执行的交换机命令


点击 “Run.exe”开始运行



提示：
采集输出的配置文件保存在Backup文件夹内

日志输出保存在log文件夹内
